# astolFOSS

Hyprland setup script for Arch Linux.

Installs:

* Hyprland
* Ly login manager
* Waybar
* PipeWire audio
* LibreWolf browser
* Fish shell
* Paru AUR helper
* Astolfo wallpaper
* Waybar config

## Install
```
sudo pacman -S --needed git base-devel
git clone https://github.com/kaiiress/astolFOSS.git
cd astolFOSS
chmod +x install.sh
./install.sh
```
